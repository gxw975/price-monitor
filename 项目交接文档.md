## 7. 废弃路径（已否决方案，禁止重提）

| # | 方案 | 原因 |
|----|------|------|
| D1 | 用 `"msgtype"` 作为飞书字段名 | API v2 要求 `"msg_type"`（带下划线） |
| D2 | 用 `urllib.request` 发 HTTP 请求 | 改用 `requests` 库，错误处理更好 |
| D3 | 直接 `"opencli"` 字符串调用 | Supervisor 环境无 nvm PATH，必须用 `/usr/local/bin/opencli` |
| D4 | 绝对路径写死 Python 解释器 | 改用 `cd + ./venv/bin/python` 相对路径 |
| D5 | crontab 中用绝对路径 | 改用 `cd /path && ./venv/bin/python` |
| D6 | opencli `screenshot --output` 语法 | opencli 不支持 `--output`，正确语法是 `screenshot <path>` |
| D7 | opencli `browser <session> status` | 正确命令是 `browser <session> state` |
| D8 | 两个 venv 混用 | `.venv`（3.11.15）用于 Supervisor，`venv`（3.12.3）用于 crontab，不能混用 |
| D9 | 设置标签页的通道切换立即保存 | 已改为按钮保存（draftAlert 模式） |
